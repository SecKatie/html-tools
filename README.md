# HTML Tools

A collection of single-file HTML utilities written by LLMs.

## Available Tools

| Tool | Description |
|------|-------------|
| [Hacker News Comment Search](hn-comments.html) | Search for any URL on Hacker News and view its comments and discussion threads |

## Usage

Visit the live site at: **https://seckatie.github.io/html-tools/**

Or clone the repo and open any HTML file directly in your browser - no build step or server required.

## Contributing

Each tool should be a single, self-contained HTML file with inline CSS and JavaScript. No external dependencies.
